<objective>
Extend renderer/render.py to export high-bit-depth 4K image sequences by adding resolution configuration, camera orbit animation, Cycles optimization, and the render execution call.
</objective>

<must_haves>
- Script MUST set render resolution to 3840×2160 (4K landscape).
- Script MUST configure PNG 16-bit output with sequential frame naming (`frame_####`).
- Script MUST add camera orbit animation keyframes (turntable rotation over the frame range).
- Script MUST call `bpy.ops.render.render(animation=True)` to produce the frame sequence.
- Script MUST accept `--frames N` and `--output-dir PATH` command-line arguments.
- Rendered frames MUST appear in the specified output directory with zero-padded numbering.
</must_haves>

<action>
Modify `renderer/render.py` to add the following after the existing camera setup:

1. **Parse new CLI arguments** after the existing `--camera` parsing block:
   ```python
   frame_count = 300
   if "--frames" in argv:
       try: frame_count = int(argv[argv.index("--frames") + 1])
       except (IndexError, ValueError): pass

   output_dir = "renders/frames"
   if "--output-dir" in argv:
       try: output_dir = argv[argv.index("--output-dir") + 1]
       except IndexError: pass

   output_format = "PNG"
   if "--format" in argv:
       try: output_format = argv[argv.index("--format") + 1].upper()
       except IndexError: pass
   ```

2. **Set 4K resolution** (add before render execution):
   ```python
   scene = bpy.context.scene
   scene.render.resolution_x = 3840
   scene.render.resolution_y = 2160
   scene.render.resolution_percentage = 100
   ```

3. **Configure output format** — PNG 16-bit by default, TIFF option:
   ```python
   import os as _os
   _os.makedirs(output_dir, exist_ok=True)
   scene.render.filepath = _os.path.join(output_dir, "frame_")
   if output_format == "TIFF":
       scene.render.image_settings.file_format = 'TIFF'
       scene.render.image_settings.color_depth = '16'
   else:
       scene.render.image_settings.file_format = 'PNG'
       scene.render.image_settings.color_depth = '16'
       scene.render.image_settings.color_mode = 'RGBA'
       scene.render.image_settings.compression = 15
   ```

4. **Cycles optimization**:
   ```python
   scene.cycles.samples = 128
   scene.cycles.use_denoising = True
   scene.cycles.denoiser = 'OPENIMAGEDENOISE'
   ```

5. **Camera turntable animation** — after the existing Track To constraint code:
   ```python
   scene.frame_start = 1
   scene.frame_end = frame_count
   target.rotation_euler = (0, 0, 0)
   target.keyframe_insert(data_path="rotation_euler", frame=1)
   import math
   target.rotation_euler = (0, 0, math.radians(360))
   target.keyframe_insert(data_path="rotation_euler", frame=frame_count)
   # Set linear interpolation for smooth constant-speed rotation
   if target.animation_data and target.animation_data.action:
       for fcurve in target.animation_data.action.fcurves:
           for kf in fcurve.keyframe_points:
               kf.interpolation = 'LINEAR'
   ```

6. **Execute render**:
   ```python
   print(f"Rendering {frame_count} frames at 3840x2160 to {output_dir}...")
   bpy.ops.render.render(animation=True)
   print(f"Render complete. {frame_count} frames written to {output_dir}")
   ```
